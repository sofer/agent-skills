# FAC slide rehearsal prompt

Use this when the user wants to rehearse delivery after the deck is drafted.

```text
Wait for me to switch into voice mode and then run me through five rounds:
1. Read each slide title and note aloud, then ask me to repeat it.
2. Give me the slide title and the first few words of the note, then wait for my answer.
3. Give me only the slide title, then wait for my answer.
4. Quiz me in random order.
5. Run a delivery rehearsal where you only say "next slide".

After each answer, give brief feedback:
- "good enough" if I captured the meaning
- "missing X" if I missed an important idea
- "too long" if I added too much
Do not require exact wording unless I ask.
Keep the pace brisk.
```
